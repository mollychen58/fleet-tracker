===============================================================================
                              DATA FLOW DIAGRAM
===============================================================================

READ FLOW (Query Operations)
=============================

User Action (Click/Search/Filter)
        |
        v
Component State Update (useState)
        |
        v
Service Layer Function Call
        |
        v
Data Layer Access (In-Memory Store)
        |
        v
Return Data
        |
        v
Component Re-render
        |
        v
Updated UI Displayed to User


WRITE FLOW (Mutation Operations)
=================================

User Action (Form Submit/Status Change)
        |
        v
Event Handler Triggered
        |
        v
Service Layer Mutation Function
        |
        v
Validation (Business Rules)
        |
        +---> [FAIL] Return {success: false, error: "..."}
        |                   |
        |                   v
        |             Display Error Message
        |
        +---> [PASS] Update In-Memory Store
                      |
                      v
                Return {success: true}
                      |
                      v
                Set Success State
                      |
                      v
                Component Re-render
                      |
                      v
                Display Success Message + Updated Data


===============================================================================
                            KEY USER INTERACTIONS
===============================================================================

1. SEARCH & FILTER
   User Input --> searchVehicles(query) / getVehicles(filter) --> Results

2. SELECT VEHICLE
   Row Click --> Set selectedVehicleId --> Load Detail Data --> Show Panel

3. SCHEDULE MAINTENANCE
   Form Submit --> scheduleMaintenance() --> Validate --> Create Record

4. UPDATE STATUS
   Dropdown Change --> updateVehicleStatus() --> Validate --> Update Vehicle

5. RESET DATA
   Button Click --> resetData() --> Clear Stores --> Reset UI State

6. TOGGLE VIEW
   Button Click --> Set viewMode --> Render Table or Cards
